# Usage

<!-- usage -->

```sh-session
$ npm install -g chilli-cloud-code-cli
$ chilli-cloud-code COMMAND
running command...
$ chilli-cloud-code (-v|--version|version)
chilli-cloud-code-cli/0.0.1 darwin-x64 node-v13.11.0
$ chilli-cloud-code --help [COMMAND]
USAGE
  $ chilli-cloud-code COMMAND
...
```
